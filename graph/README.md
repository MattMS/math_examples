# Graph

A set of vertices connected by edges.


## Descriptions

### Graph

"Order" is the number of vertices.
"size" is the number of edges.


### Vertex

AKA: nodes, points, vertices.

"Degree" or "valency" in the number of connected edges.


### Edge

AKA: arcs, edges, lines, links.
