# Logarithm

If `8` equals the base `2` raised to the exponent/power `v`, then `v` is
the logarithm of `8` to base `2`.
So `v` is `3`.

In CoffeeScript:

```coffee-script
v = 3
8 == Math.pow 2, v
v == Math.log2 8
```
